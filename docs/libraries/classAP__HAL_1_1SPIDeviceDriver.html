Member Enumeration Documentation

enum AP_HAL::SPIDeviceDriver::bus_speed

optional set_bus_speed() interface. This can be used by drivers to request higher speed for sensor registers once the sensor is initialised. This is used by the MPU6000 driver which can handle 20MHz for sensor register reads, but only 1MHz for other registers.
